Lab 7

Markov Chains

When modern browsers like Chrome visit a webpage, there is sometimes an option to automatically translate the text into English from another language. How does the browser detect other languages? One way is to pass the text of the webpage through prebuilt Markov chains for many languages, and determine which language has the highest probability of generating this text.

Download the four classic books above, and learn the probabilities for a Markov chain that models the adjacent letter patterns found in each of the given languages. We will ignore the header and footer text in the files, only focus on the actual book text. Your states in the Markov chain will be each possible letter in any of the four languages and the space character. The Markov chain should be fully connected, so that any state can be followed by any other state. Use the concept of a pseudocount to initialize every possible observation to be 1, then increment these counts for the adjacent pairs found in the text, and finally normalize your counts to be probabilities. For simplicity, ignore all punctuation and capitalization. You will need to incorporate log probabilities.
